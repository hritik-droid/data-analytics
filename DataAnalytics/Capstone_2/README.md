# Capstone 2 - Business Analysis with Python

## Project Overview
This project analyzes sales data for EmporiUm, a fictional bookstore and retail chain. The analysis uses Python, pandas, NumPy, and Matplotlib to evaluate sales performance, customer behavior, and product category trends.

## Files Included
- zope_sales_analysis.ipynb
- StoreSales.csv
- StoreDetail.csv
- Products.csv
- ProductCategories.csv
- customer_list.csv

## Analysis Included
- Territory manager analysis
- Monthly revenue trends
- Store performance rankings
- Top customer analysis
- Product category revenue analysis
- Marketing recommendations
- Matplotlib visualizations

## Technologies Used
- Python
- pandas
- NumPy
- Matplotlib
- Jupyter Notebook
